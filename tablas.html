Tablas

Son elementos que utilizaremos en nuestra pagina para poder tener informacion agrupada, independientemente del tipo de datos que sea. para ellos usaremos la etiqueta table.

Tablas Simples

Se conforman de tres elementos que se engloban uno dentro de otros. el primer elemento importe es table.

<table> ...</table>

En segundo lugar nos encontraremos con el elemento tr, que nos determina las filas que tendran nuestra tabla.

  1. <table>
  2. <tr>...</tr>
  3. <tr>...</tr>
  4. <tr>...</tr>
  5. </table>

Por ultimo tendremos el elemento td, el cual hace referencia a las celdas que tendremos en nuetsra tabla. Lo que escribamos entre las etiquetas td será el contenido que estara reflejado en cada una de las celdas

  1. <table>
  2. <tr>
  3. <td>...e1...</td>
  4. <td>...e2...</td>
  5. <td>...e3...</td>
  6. </tr>
  7. </table>
...e1...
...e2...
...e3...

Tablas básicas

Como ya sabemos hacer una tabla simple, podemos añadir algunos elementos ams que nos ayudaran a crear las tablas como deseamos.

Para ello, lo primero que haremos sera bautizar a nuestra tabla. Ponerle un titulo. Usaremos la etiqueta caption.

  1. <table>
  2. < caption>Ejemplo</caption>
  3. <tr>
  4. <td>...e1...</td>
  5. <td>...e2...</td>
  6. <td>...e3...</td>
  7. </tr>
  8. </table>

Cuando ya tenga nuestra tabla nombre, podemos crear un resumen de los datos contenido en ella. Lo haremos a traves del atributo summary.Aunque no lo parezca es un atributo de gran importancia ya que las paginas web seran interpretadas por aplicaciones y programas para personas con discapacidad, y por lo general los datos o la informacion que contendra nuestra tabla puede ser complicado de interpretar.

Su sintesxis de código es:

  1. <table summary:"Tabla con datos ejemplo">
  2. < caption>Ejemplo</caption>
  3. <tr>
  4. <td>...e1...</td>
  5. <td>...e2...</td>

Otros atributos

Hay atributos que nos especifican aun mas las cabeceras, o nos ayudan a adarle a la tabla un formato mas acorde con lo que queremos, agrupando celdas. es e caso de los atributos scope y colspan.

Scope nos ayuda a explicar la informacion que hay en lsa filas o columnas de nuestra tabla. se ubica en la primer celda, y lo haremos añadiendo al elemento td el atributo scope=" col", para columnas, "row, para filas, o "colgroup", para todas las columnas o "rowgroup", para todas las filas.

  1. <td scope="col">contenido random de la celda</td>

Colspan nos permite agrupar celdas como podemos hacer en una excel normal.Sun funcionamiento es el mismo, ya que debemos saber cuantos espacion ocupara nuestras celdas. este atributo uira siempre el el elemento td que queramos combinar.

Ejemplo:

  1. <td colspans="2">contenido de la celda</td>

Tablas avanzadas

Cuando ya dominemos las tablas simples y avanzadas, podremos pasar a complicar un poco mas las tablas en HTML, para ello podemos empezar con las tablas avanzadas. Estas básicamente cubren el resto de opciones que las tablas básicas no pueden. Ayudar a usuarios no visibles, determinar el ancho de las tablas...

En primer lugar, en nuestras tablas avanzadas podremos crear grupos de filas a través el elemento tr. Las filas las podemos agrupar en 3 partes:

  1. Cabeceras, a través del elemento thead.
  2. Cuerpo, con el elemento tbody.
  3. Pie, con el elemento tfoot.

Cada elemento tendrá una o más filas, en función de lo que queramos agrupar.La estructura seria

  1. <thead>
  2. <tr>...</tr>
  3. </thead>
  4. <tbody>
  5. <tr>...</tr>
  6. </tbody>
  7. <tfoot>
  8. <tr>...</tr>
  9. </tfoot>

    10. Si bien es recomendable ponerlas en ese orden siempre, que es como lo visualizara e interpretara el navegador, no es necesario.

    También podemos agrupar las columnas con relación semántica. Para ello usaremos el elemento colgroup con el atributo span definiendo además el ancho de las columnas con width.

    En el atributo span definiremos el numero de columnas que representa las agrupación, de izquierda a derecha. con el atributo width>/b>conseguiremos especificar el ancho para la columna agrupada.

    El código quedaría:

    <colgroup span="numeroColumnas" width="anchoColumna">

    En caso de querer manipular los estilos gráficos de las columnas, podemos recurrir a usar el elemento col, que se alojaría dentro del elemento colgroup.

    el atributo que definiremos con el elemento col, será el atributo width.
